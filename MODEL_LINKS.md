## Notes

- **Vision models** require both the main model and the mmproj (vision projector) file
- **LiteRT (.litertlm/.task)** models use Google's LiteRT-LM runtime
- **GGUF models** use llama.cpp for inference
- All models are stored in the app's internal storage under `/data/data/com.quantlm.yaser/files/models/`
